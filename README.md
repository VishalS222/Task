Assignment Dashboard – Full-Stack Demo

A lightweight full-stack dashboard application showcasing authentication, CRUD operations, and scalable project structure.
Built with React on the frontend and Node.js on the backend.
Note: The backend is hosted on Render and may take a few seconds to wake up. You might need 2–5 initial attempts before the first request responds. The frontend is deployed on Vercel.


Features
Dashboard
Displays authenticated user profile (fetched from backend)
CRUD operations on a sample entity
Search and filter UI for quick data access
Logout flow with token clearance

Security & Scalability
Password hashing using bcrypt
JWT-based authentication and middleware protection
Centralised error handling and request validation
Modular and scalable folder structure for future expansion


Tech Stack
Frontend: React
Backend: Node.js 
Auth: JWT
Data Handling: REST APIs
